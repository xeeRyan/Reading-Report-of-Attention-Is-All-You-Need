## 📖 报告简介

本报告是个人对 Google Brain 团队 2017 年经典论文 **《Attention Is All You Need》** 的深度阅读与系统性梳理。

报告从问题动机出发，详细解析 Transformer 的架构设计（自注意力、多头注意力、位置编码等），分析其有效性的深层原因，回顾其从机器翻译到统治全球 AI 建模的十年影响史，并进行批判性反思——讨论二次复杂度、位置编码缺陷、可解释性幻象等局限。

报告不仅仅是对技术的复述，更是一次对**范式转移过程**的体察，以及对研究者**方法论启示**的提炼。
